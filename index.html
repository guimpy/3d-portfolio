Sobre Mí

¡Hola de nuevo!

Soy un estudiante apasionado por el desarrollo de software y el arte digital. Este proyecto forma parte de mis materias curriculares y me permite experimentar con el flujo de trabajo completo: desde el modelado en 3D con Blender hasta la exportación para entornos web interactivos mediante WebGL.

Mi enfoque principal es combinar la lógica del código con interfaces interactivas atractivas. Disfruto construyendo mundos digitales, tanto en código plano como en entornos tridimensionales inmersivos.

Especialidad: WebGL / Blender 3D / Web Dev
Intereses: Motores de Videojuegos, UI/UX, Animación
